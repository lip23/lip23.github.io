gitpage&hexo建站

发表于 2018-12-08

本文记录该网站的建立过程，主要记录一些原理性的内容，建站的具体步骤在很多网站上都有详细介绍，这里不再重复，在最后会给几个参考的网站。

gitpage

什么是gitpage

gitpage是github为用户提供的用来介绍自己(或自己项目)的网站，网站的地址为https://username.github.io，网站的源码由用户自己提供。用户只需将网站源码上传到gitHub中名为username.github.io的项目上，然后github会帮助部署并发布网站。

如何创建gitpage

参考官方教程：https://pages.github.com/

如何自定义url

gitpage支持自定义url，主要设置操作有两步：

  • 在username.github.io项目中创建CNAME文件并在其中填上自定义的url地址
  • 在申请url的网站中找到dns服务器设置，在设置中添加github服务器的ip地址

（具体步骤可参考该网址：在github上搭建自己的主页和顶级域名的绑定）

自定义url到gitpage主页的工作原理：
当在浏览器中输入自定义url后，会首先查找dns服务器，找到该url对应的ip（即我们设置的github服务器ip），然后访问github服务器；github服务器这时会根据url查找我们设置的CNAME文件，找到对应的gitpage主页。

hexo

什么是hexo

hexo是一个网站框架，可以让我们在不了解前端知识的情况下建立一个静态网站。

如何使用hexo

参考官方教程：https://hexo.io/zh-cn/docs/

注意事项

  • 官方教程是最好的资料，强烈建议参考官方教程。
  • 在用npm安装node.js的时候若下载速度很慢可以更换淘宝镜像
    npm config set registry "https://registry.npm.taobao.org"
